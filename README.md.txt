# Aplikasi Prediksi Harga Kripto dengan Machine Learning

Ini adalah proyek Tugas Akhir saya untuk gelar Sarjana Teknik Informatika. Aplikasi ini merupakan sebuah dashboard web interaktif yang dibangun untuk memprediksi harga aset kripto Cardano (ADA) menggunakan model machine learning LightGBM.
